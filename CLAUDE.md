# Dragones Valenciana — D&D Project

## Что это

Материалы для D&D кампании в сеттинге фэнтезийной Валенсии. Проект ведёт DM (Dungeon Master), я помогаю писать приключения, NPC, локации и всё остальное. Язык контента — русский.

## Сеттинг

Фэнтезийная Испания/Валенсия. Мир "Dragones Valenciana" — с драконами на Севере, теократией на Юге, и торговой Валенсией посередине. Лор мира: `lore/`.

## Структура проекта

```
adventures/           — Приключения (ваншоты, модули), с числовым префиксом по хронологии
  001-paella-valeciana/   — Первый ваншот: "Куролиск для Paella Valenciana"
    oneshot-paella-valenciana.md  — Основной модуль (сценарий для DM)
    players/                     — Листы персонажей, PDF, арты
    game-results/                — Итоги после игры: опыт, награды, дневники
    game-report.md               — Отчёт об игре
    combat-tracker.md            — Трекер боя
characters/           — NPC (персонажи мастера), общие для всех приключений
locations/            — Локации мира
organizations/        — Организации, ордена, гильдии (подробные описания)
lore/                 — Лор мира: история, фракции, магия
raw/                  — Входящие файлы для обработки (временная папка, не в git)
homebrew.md           — Хоумбрю правила
dnd-art-style-prompt.md — Главный промпт для генерации изображений (единый арт-стиль кампании)
```

### Папка `raw/`

Временная папка-inbox. DM скидывает сюда файлы (HTML-листы с D&D Beyond, тексты, изображения и т.д.), Claude обрабатывает их: извлекает данные, создаёт/обновляет файлы в нужных местах проекта. После обработки файлы из `raw/` удаляются. Папка добавлена в `.gitignore`.

## Docsify-сайт

Проект опубликован как Docsify-сайт (GitHub Pages). Навигация через `_sidebar.md`, конфиг в `index.html`. Ссылки в markdown-файлах — относительно корня проекта (не относительно файла).

## Конвенции

- **Приключения** нумеруются: `001-name`, `002-name` и т.д. — хронологический порядок создания
- **NPC** живут в `characters/`, **локации** в `locations/` — они общие, не привязаны к одному приключению
- Каждый NPC и локация содержат ссылку "Появляется в:" на приключение
- При перемещении файлов — обязательно проверять и обновлять все ссылки (grep по старому пути)
- Контент пишется на русском, имена/названия могут быть на испанском

## Поиск приключений для адаптации

Когда DM приходит без идеи для следующего ваншота — ищем готовые приключения и адаптируем под сеттинг Dragones Valenciana.

### Источники (проверены, доступны через WebFetch)

| Сайт | Что там | URL |
|------|---------|-----|
| **Adventure Lookup** | 3400+ приключений, фильтры по системе/уровню/тегам | adventurelookup.com |
| **Kassoon** | 200+ готовых ваншотов с NPC, сценами, энкаунтерами | kassoon.com/dnd/one-shots/ |
| **Nat1 Gaming** | 120+ бесплатных приключений, сортировка по уровням | nat1gaming.com/dnd/articles/dnd-adventures/ |

DMs Guild (dmsguild.com) блокирует WebFetch — для него DM ищет вручную или через Perplexity.

### Рабочий процесс

1. **Запрос** — DM задаёт параметры: уровень партии, жанр, настроение, или просто "найди что-нибудь интересное"
2. **Поиск** — ищу по источникам выше, выдаю 5-10 вариантов с описаниями
3. **Углубление** — DM выбирает кандидата, я читаю полное описание и рецензии
4. **Адаптация** — берём в работу, переносим в сеттинг, создаём `adventures/NNN-name/`

### Параметры для поиска

- Уровень партии (текущий: 2)
- Жанр: детектив, хоррор, хейст, данжен-кроул, социалка, монстр-хант
- Сеттинг: город, дикая природа, подземелье, море, горы
- Настроение: "давно не было боёвки", "хочу загадки", "что-то мрачное"
